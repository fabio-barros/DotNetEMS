# DotNetEMS

<p> 🖥 Employee Management System wq .NET, React, Docker + Heroku. </p>

### ⚙️ Structure

- Client w/ React + Typescript, React Admin🚧
- .NET Core Server API, Docker + Heroku 👷
- Heroku Postgres DBaaS 🚧
